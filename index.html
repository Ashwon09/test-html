Quick deploy checklist

  1. Push index.html to your GitHub repo.
  2. In GitHub ‚Üí Settings ‚Üí Pages set the source branch (e.g. main).
  3. Add a CNAME file with your custom domain or set it in the Pages settings.
  4. Add DNS records at your registrar (A records for apex or CNAME for www).
  5. Wait for DNS propagation and enable HTTPS in Pages settings (if not automatic).
